Project on Console

Visual Studio is a platform that allows the user to code in many coding languages. It is developed by Microsoft.To set up Visual Studio you have to create a new file and open it in Visual Studio. To open it you have to click on open folder.

The console screen is a place where the results of functions and mathemtical opeartions can be shown. It is mainly used to check if the function works before printing it on the main screen. To open console screen press right-click and select inspect. Then right next to elements there should be console which you should click. To print something on console you need to type console.log(); and in the brackets right the value you want it to display.
